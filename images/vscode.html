In this course you will learn the #1 Javascript library for building websites: React. You will also be introduced to advanced aspects of JavaScript ES6, functional components using React Hooks and will develop state-aware applications using the latest redux toolkit.

Duration:

5 Weeks + Project Portfolio

10-14 hours/week online lectures and exercises, 4 hours/week live online workshop and assignments.

Using React Native and the Expo SDK to access all native hardware capabilities of modern phones and tablets.

Duration:

4 Weeks + Project Portfolio

Commitment:

6 hours/week online lectures and exercises, 4 hours/week live online workshop and assignments.

Duration:

4 Weeks + Project Portfolio
